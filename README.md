######## COMANDOS BASICOS LINUX ##########

(ls) lista directorios.
(pwd) Muestra ruta actual.
(ls -al) Muestra informacion de archivos y directorios.
(ls - al [nombre archivo.txt]) Muestra información de un archivo o directorio especificado dentro de los paréntesis cuadrados.
(touch [texto.txt]) Crea un archivo con nombre y extensión del archivo especificado dentro del paréntesis cuadrados.
(cp [texto.txt] [texto2.txt]) Nos permite copiar un archivo y asignarle un nombre nuevo.
(cd ..) Nos permite retornar a la ruta que estábamos antes.
(cp -r [Downloads/] [copia]) Nos permite hacer una compia de un directorio.
(mv [archivo.txt] [archivo-movido.txt]) Nos movera archivos entre archivos.
(mkdir [Linux] ) Nos permite crear directorios.
(rmdir [Linux]) Eliminara un directorio (SOLO FUNCIONARA SI EL DIRECTORIO SE ENCUENTRA VACIO).
(rm [archivo.txt]) Eliminara un archivo.
(rm -r [Carpeta_downloads/]) Nos eliminara un directorio con archivos.  
(ip a) muestra información detallada sobre todas las interfaces de red del sistema, incluyendo direcciones IP y estado de las interfaces.
